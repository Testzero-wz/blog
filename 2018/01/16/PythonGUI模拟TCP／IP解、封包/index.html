PythonGUI模拟TCP／IP解、封包

发表于 2018-01-16 | 分类于 安全之旅

认真学习了TCP/IP中包的结构以及封包解包，分包重组等步骤
利用实验之余写一个比较严格执行TCP/IP模型的解、封包模拟程序并以仿wireshark的GUI形式展现

本程序使用5层TCP/IP模型(多了个物理层)

比较严格地实现了

  • 各个层的封包解包规则以及包结构
  • 任意字符发包
  • 动态构建菜单
  • 数据包长度控制
  • 网络层乱序包排序重组
  • 每层的包数据含义解析
  • 十六进制数据展示

每个层的结构，源码里面有展示
GUI库用的是Python自带的tkinter，不需要安装，有python2.7环境就能运行
